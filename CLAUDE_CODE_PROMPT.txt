I need you to build a complete 3D restaurant food scanning and ordering system. This is an INTERMEDIATE level project with the following requirements:

## PROJECT OVERVIEW
Build a full-stack restaurant system where:
1. Staff can upload 3D models of food (GLB/GLTF format) 
2. Customers can view interactive 3D models of menu items before ordering
3. System works on web, mobile, and kiosk displays
4. Includes AR preview capabilities

## TECHNICAL STACK
- **Backend**: Node.js + Express
- **Database**: PostgreSQL for menu data, file storage for 3D models
- **Frontend**: React with Three.js for 3D rendering
- **3D Format**: GLB/GLTF (industry standard for web 3D)
- **AR**: model-viewer library (supports ARKit and ARCore)

## FEATURES TO IMPLEMENT

### 1. BACKEND API (Node.js/Express)
Create RESTful API with these endpoints:

**Menu Management:**
- POST /api/menu - Upload new dish with 3D model
- GET /api/menu - Get all menu items (with filtering by category)
- GET /api/menu/:id - Get single menu item details
- PUT /api/menu/:id - Update menu item
- DELETE /api/menu/:id - Delete menu item
- GET /api/categories - Get all food categories

**File Upload:**
- Support uploading .glb, .gltf files (up to 50MB)
- Support uploading thumbnail images (JPG/PNG)
- Store files in organized folder structure
- Generate unique filenames to prevent conflicts

**Search & Filter:**
- GET /api/search?q=query - Search menu items by name/description
- GET /api/menu?category=appetizer - Filter by category
- GET /api/menu?priceRange=10-20 - Filter by price range

**Database Schema (PostgreSQL):**
```sql
CREATE TABLE menu_items (
  id SERIAL PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  category VARCHAR(100),
  description TEXT,
  price DECIMAL(10,2),
  currency VARCHAR(3) DEFAULT 'USD',
  model_path VARCHAR(500),
  thumbnail_path VARCHAR(500),
  allergens JSONB,
  nutritional_info JSONB,
  available BOOLEAN DEFAULT true,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);

CREATE TABLE categories (
  id SERIAL PRIMARY KEY,
  name VARCHAR(100) UNIQUE NOT NULL,
  display_order INT
);

CREATE TABLE orders (
  id SERIAL PRIMARY KEY,
  table_number INT,
  items JSONB,
  total_price DECIMAL(10,2),
  status VARCHAR(50),
  created_at TIMESTAMP DEFAULT NOW()
);
```
